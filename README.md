# 🎓 Student Management System

## 📌 Project Overview

This is a web-based Student Management System developed using Spring Boot.
It performs CRUD operations (Create, Read, Update, Delete) on student data.

---

## 🚀 Features

* ➕ Add Student
* ✏️ Update Student
* ❌ Delete Student
* 📋 View All Students

---

## 🛠️ Tech Stack

* Backend: Java, Spring Boot
* Frontend: Thymeleaf, HTML, CSS
* Database: MySQL
* Build Tool: Maven

---

## 📂 Project Structure

* src/main/java → Java source code
* src/main/resources/templates → HTML pages
* application.properties → Configuration

---

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Run `StudentManagementSystemApplication.java`
4. Open browser: http://localhost:8080/students
